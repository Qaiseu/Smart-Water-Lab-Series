# Smart-Water-Lab-Series

Laboratory projects developed for the Smart Water Lab Series at Xi’an Jiaotong University -2026, focusing on hydrological modeling, flood analysis, rainfall forecasting, and water resources optimization. Each experiment incorporates AI-assisted development workflows and includes documented AI interaction records in a `prompt_log.md` file.

## Projects

### Rainfall-Runoff Time Series
A rainfall forecasting and alert system developed using Python and Streamlit to monitor rainfall trends, visualize weather data, and provide basic rainfall intensity alerts for different locations.
https://github.com/Qaiseu/Rainfall-Forecasting-Alert-System

### Hydrological Modeling: SCS-CN Runoff Calculation
A hydrological modeling project implementing the SCS-CN method to estimate surface runoff from rainfall data, including runoff computation, hydrograph generation, and rainfall-runoff analysis.
https://github.com/Qaiseu/Hydrological-Modeling

### Water Resources Optimization: Reservoir Dispatch
A reservoir operation and water allocation optimization model designed to analyze storage behavior, water shortages, and system reliability using optimization and simulation techniques.
https://github.com/Qaiseu/Water-Resources-Optimization

### Flood Inundation Analysis
A DEM-based flood inundation analysis project for simulating flood extent, estimating inundated areas and water volume, and visualizing flood propagation under different water levels.
https://github.com/Qaiseu/Flood-Inundation-Analysis
